About

This tool safely checks to see if your password is in a huge database of passwords used by hackers. If it is, your password is definitely NOT safe and you should change it immediately!

This website is safe to use because we never even know your password! Read 'How it Works' to learn more.

How it Works

When you enter your password, your browser hashes it using javascript.

Hashing is a one-way mathematical function that always results in the same output. That output can't be reverse engineered to determine the input (that's why it's called one-way). Learn more about hashes here.

After your browser hashes your password and you click the green 'check it' button, your browser sends the hash (not the password) to us. We look up the hash in our database that contains the hashes of over 1.5 billion passwords. If it's there, you know that your password is NOT secure.

Using the hashed value allows us to tell you wether or not your password is in the database, without even knowing your password!

If you're really paranoid, look at our source code, or use our API!

API

If you want to check passwords programmatically, use the following URL:

http://checkthis.pw/pwcheck.php/?hash={sha1hash}

*Where {sha1hash} is the SHA1 hash of the password you want to check.

We will respond with a 1 if the hash is in our database, and a 0 if it is not. We encourage you to use this free service if:

  • you're signing up new users for your website
  • you want to check a bunch of passwords
  • you don't trust us and are too lazy to read the source code ;-)
